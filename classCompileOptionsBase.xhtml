Detailed Description

This is a base class for all classes needing compilation parameters. It offers functions to get/set the following:

  • Compiler options
  • Linker options
  • Compiler include dirs
  • Resource compiler include dirs
  • Linker include dirs
  • Custom commands to be executed before/after build
  • The settings modification status

    These settings are used by the compiler plugins to construct the necessary compilation commands.
