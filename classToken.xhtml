Detailed Description

a symbol found in the parsed files, it can be many kinds, such as a variable, a class and so on. Once constructed, the Token should be always put in the TokenTree, and we can access the Token by TokenIndex.

int Token::m_Index

current Token index in the tree, it is index of the std::vector<Token*>, so use the index, we can first get a address of the Token, and after a dereference, we can get the Token instance. Note that the tree is a dynamic object, which means Tokens can be added or removed from the tree, so the slot of std::vector<Token*> is reused. So, same index does not point to the same Token if TokenTree are updated or changed.
